# 47 Sunset Studios Landing Site

A dark-themed landing page for 47 Sunset Studios, showcasing digital solutions.

## Features

- Vague, mysterious description of digital solutions
- Dark theme aesthetic
- Hosted on Cloudflare
- Role-based views (future enhancement)
- Integration with campaign builder (future enhancement)

## Getting Started

Clone the repository and navigate to the folder:

```bash
git clone https://github.com/goblinsan/47-sunset-studios-landing-site.git
cd 47-sunset-studios-landing-site
```

Install dependencies (if any):

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## Deployment

This project is deployed on Cloudflare Pages. To deploy:

1. Push to the `main` branch
2. Configure Cloudflare DNS to point to the deployed URL
3. Set up SSL/TLS
